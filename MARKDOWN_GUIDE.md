# Communicate Using Markdown

## What is Markdown?

Markdown is a **lightweight markup language** for formatting text. Here's a showcase:

### Headers
Use `#` for headers — `#` for H1, `##` for H2, etc.

### Lists
- Unordered lists use `-` or `*`
1. Ordered lists use numbers

### Code
Inline `code` uses backticks. Code blocks use triple backticks:

```python
def hello_world():
    print("Hello, GitHub!")
```
