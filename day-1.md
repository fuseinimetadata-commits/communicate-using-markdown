## Day 1: Introduction to Markdown

Today I'm learning about **Markdown**, a lightweight markup language for creating formatted text.

## What I Learned

Markdown uses simple symbols to create formatting:

- Headings use `#` symbols
- Lists use `-` or `*`
- **Bold** uses double asterisks
- _Italic_ uses underscores
